## File Naming Conventions

### Images
- Use lowercase with hyphens: `client-meeting.jpg`
- Include project name: `mercury-campaign-2024.jpg`
- Add version numbers: `logo-v2.png`

### Videos
- Use descriptive names: `product-demo-30s.mp4`
- Include duration: `brand-story-60s.mov`
- Add quality indicators: `high-quality-4k.mp4`

### Documents
- Use clear naming: `contract-template-2024.pdf`
- Include dates: `invoice-jan-2024.pdf`
- Add status: `script-approved-v2.pdf`

## Supported File Formats

### Images
- **JPG/JPEG** - For photographs and complex images
- **PNG** - For graphics with transparency
- **WebP** - For web optimization (recommended)
- **SVG** - For scalable graphics and icons

### Videos
- **MP4** - Most compatible format
- **MOV** - High quality, larger files
- **AVI** - Legacy format support
- **WebM** - Web-optimized format

### Documents
- **PDF** - For contracts and documents
- **DOC/DOCX** - For editable documents
- **TXT** - For simple text files

## Image Guidelines

### Recommended Sizes
- **Hero Images**: 1920x1080px (16:9)
- **Thumbnails**: 400x300px (4:3)
- **Portfolio**: 800x600px (4:3)
- **Team Photos**: 600x600px (1:1)
- **Icons**: 64x64px or 128x128px

### Quality Standards
- **Web Optimization**: Compress for fast loading
- **High Resolution**: Maintain quality for different screen sizes
- **Brand Consistency**: Use Mercury Solutions color palette
- **Professional Quality**: Ensure images represent brand standards

## Usage Instructions

1. **Upload Files**: Place files in the appropriate subdirectory

3. **Optimization**: Compress images for web use
4. **Organization**: Keep related files together
5. **Backup**: Maintain backups of original files

The application will automatically:
- Detect and display images based on file names
- Optimize images for different screen sizes
